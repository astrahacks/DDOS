### Installing

* `pkg update`
* `pkg upgrade`
* `pkg install git`
* `apt install python`

* `apt install python3`
* `git clone https://github.com/astrahacks/DDOS.git`

#### Finished you have successfully downloaded DDos now to launch!

* `cd DDos`
* `python2 ddos.py`
